# Run the project using docker

### after cloning the repository change the dierctory
>> `cd project`
>> 
>> `touch .env`

check if you have docker installed
### run this command to execute the docker container
>> `docker compose up --build`

## To run the frontend project
from the root of the project (`project/cloudtryx_assignment_frontend/`)
>> `npm install`
>> `npm run dev`
